#### 1.3.1 TransMac 重置试用

&emsp;&emsp;注册表删除 HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Shell Extensions\Approved 这个 Key就可以，这里提供 cmd 脚本：

```cmd
@echo off
reg delete "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Shell Extensions\Approved" /f
echo 重置 TransMac 试用完成
pause
```

## 3.2 VMware Workstation 15 解锁 Mac OS

&emsp;&emsp;下载执行 unlocker 3，管理员运行 unlocker.exe ，等 patch OK了关掉就行。用来配置 opencore 和封装镜像。


## 3.3 VMware Workstation 15.5.5 CapsLock 大写锁键修复

&emsp;&emsp;在虚拟机xmv配置文件下添加如下配置：

```
mks.win32.useInjectedMagic = "FALSE"
```

&emsp;&emsp;问题出在那个增强型键盘驱动程序，没啥需要的话安装 VMware 时不要勾选它。
